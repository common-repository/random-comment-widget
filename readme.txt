== Description ==

Random Comment Widget displays random comment from selected page or post on your website. Great solution for testimonial.

== Installation ==

1. Upload random-comment-widget /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Drag the widget to your sidebar and provide the details and save the widget.
